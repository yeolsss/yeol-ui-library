# 나만의 UI Repository

## 사용한 library
- tailwindcss
  - styling library
- tailwind merge
  - Tailwind CSS 유틸리티 클래스를 효율적으로 병합하는 라이브러리
- clsx
- cva
```bash
# tailwind merge
npm install tailwind-merge
yarn add tailwind-merge

# clsx
npm install clsx
yarn add clsx

# cva(class-validator-authority)
npm install class-validator-authority
yarn add class-validator-authority
```
